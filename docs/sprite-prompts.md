# FFS Sprite Generation Guide - Leonardo.AI

## Leonardo.AI Settings (Use These Every Time)
- **Model:** Leonardo Kino XL or Phoenix (best for 3D renders)
- **Transparency:** ON (toggle in generation settings)
- **Resolution:** 512x512
- **Guidance Scale:** 7-9 (higher = more prompt adherence)
- **Number of Images:** 4 per prompt (pick the best)
- **Seed:** Lock the seed once you find a style you like, reuse it for all remaining sprites

## Style Direction
Soft 3D claymation. Think Pixar meets Play-Doh. Rounded shapes, warm lighting, gentle shadows. Premium feel, safe for kids, instantly readable at small sizes.

## Negative Prompt (Use With Every Generation)
```
text, watermark, signature, blurry, low quality, scary, dark, realistic photo, flat 2d, pixel art, anime, sharp edges, wireframe
```

## Workflow
1. Generate Batch 1 (8 style tests)
2. CJ picks the style winner
3. Lock the seed from the winner
4. Generate Batches 2-5 reusing that seed
5. Save all PNGs to `public/sprites/[game-name]/`
6. Swap in Phaser scenes (emoji Text -> Image objects)
