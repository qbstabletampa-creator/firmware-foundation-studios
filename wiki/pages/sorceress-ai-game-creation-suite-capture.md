# Sorceress AI Game Creation Suite Capture

**Summary:** Sorceress is an AI game creation suite and early access AI native game engine for indie devs and vibe coders. It combines asset tools for sprites, 3D, voxel, tilesets, audio, image, video, materials, publishing, and a WizardGenie game engine that can write and run game code in the browser. For Firmware Foundation Studios, this is a source to study for art pipeline ideas, tool packaging, and how a game factory can present many small tools under one clear studio promise.

## Notes

Sorceress positions itself as an all in one AI game creation suite:

- Asset Creation Studio for 2D and 3D assets, sound, images, video, tilesets, and more.
- WizardGenie, an AI native game engine with a vibe coding approach.
- 34 plus tools on the homepage, with a full catalog around 30 game dev tools.
- 100 free starter credits for trial users.
- Early supporter pricing: $49 one time for lifetime Pro access to non AI tools.
- Optional AI credit packs: $10 per month for 1K credits, $20 for 2K, $50 for 5K, $100 for 10K.

Key modules called out:

- Auto Sprite v2: prompt to character, AI video animation, then sprite sheet export.
- 3D Studio: image generation, image to 3D, auto rigging, weight paint refinement, text to animation, FBX, GLB, or GLTF export.
- Voxel Studio: text, image, or 3D input to rigged voxel creations.
- True Pixel and Pixel Snap: image or video to real pixel art and sprite sheets.
- Tileset Forge: AI art to aligned game ready tilesets.
- Sound Studio: music, SFX, AI voice, voice cloning, editing.
- Material Forge: image to PBR material maps.
- Procedural Walk: auto rig and IK walking for multi legged creatures.
- Storyboards: prompt to storyboard sections, then rendered video sequences.
- Publishing, Play Arcade, Layout Preview, and Marketplace are included as free or platform tools.

## Why It Matters

This directly connects to FFS because CJ is building a Christian kids game factory, not one isolated game.

The biggest useful pattern is packaging. Sorceress makes a wide toolset feel simple by grouping tools around clear jobs:

1. Make the asset.
2. Animate it.
3. Export it.
4. Test it.
5. Publish it.

That is the same shape FFS needs internally, even if CJ never buys Sorceress.
